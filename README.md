# Number Game 🔢

### 👨‍💻 Developer Details
* **Name:** Ishant Khare
* **Company:** CodSoft

### 🎯 Objective
To build an interactive console-based guessing game where the user attempts to guess a randomly generated number within a limited number of attempts.

### 🛠️ Tools Used
* Java
* Object-Oriented Programming

### 📝 Steps Performed
1. Implemented Java's random number generation.
2. Created a continuous loop to accept user guesses and provide "too high" or "too low" feedback.
3. Added a score-tracking mechanism and an option to play multiple rounds.

### 💡 Outcome
A fully functional Java console game that handles user input safely and applies robust logical control flow.
